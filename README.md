# OpenStreetMap
OpenStreetMap

การคำนวณเส้นทางและระยะทางด้วย OSM + OSRM + Leaflet

Stack ที่ใช้

OpenStreetMap (OSM): แหล่งข้อมูลแผนที่ฟรีแบบโอเพ่นซอร์ส

Leaflet.js: ไลบรารี JavaScript สำหรับแสดงผลแผนที่บนเว็บไซต์

OSRM (Open Source Routing Machine): บริการสำหรับคำนวณเส้นทางการเดินทาง (Routing Engine)

Nominatim: API สำหรับแปลงพิกัดเป็นชื่อสถานที่ (Reverse Geocoding)
